概述#

Jupyter Notebook 发布

用于编译 Jupyter 笔记本到高质量文档格式的 Sphinx 和 Docutils 扩展。

快速入门

Write

混合 Jupyter 笔记本与基于文本的笔记本，Markdown 和 RST 文档。使用 MyST Markdown 语法来支持技术创作功能，如交叉引用、图形和警告。

Compute

使用 Jupyter kernel 生成动态输出，带有可配置的执行处理。高速缓存执行输出，用于快速重建。

了解更多 »

Render

将 Jupyter 执行输出转换为丰富的嵌入式内容。在文档流中插入已计算的变量。将单个或集合的文档构建为多种格式（HTML、PDF 等）。

了解更多 »

MyST-NB 是 可执行图书项目(Executable Books Project) 中的模块，该项目是国际合作项目，旨在构建开源工具，使用 Jupyter 生态系统来促进出版计算叙事。它也是 Jupyter Book 的核心组成部分。

查看 Jupyter 图书画廊，寻找来自整个社区的灵感。

另外，还有 MyST-Markdown VS Code 扩展和 jupyterlab-myst，这些编辑工具可以用来编辑你的笔记本。
