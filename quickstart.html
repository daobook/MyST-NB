快速入门

快速入门#

myst-nb 作为 Python 包发布，不需要任何非 Python 依赖。

可以使用 pip 安装 myst-nb：

pip install myst-nb

你可以使用 mystnb-quickstart 命令来快速创建 Sphinx + MyST-NB 项目示例：

mystnb-quickstart my_project/

或者简单地将 myst_nb 添加到现有的 Sphinx 配置中：

extensions = [
    ...,
    "myst_nb"
]

默认情况下，MyST-NB 将解析 markdown（.md）和 notebook（.ipynb）。

Begin authoring your content

一旦完成了内容的创作，可以使用 sphinx-build CLI 来构建文档，例如

sphinx-build -nW --keep-going -b html docs/ docs/_build/html

小技巧

MyST-NB 是并行友好型的，所以也可以使用 sphinx-build -j 4 在 N 个进程上分发构建（和笔记本的执行）。
